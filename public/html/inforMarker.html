Format Requirements( version 1.0 )


Upload File Format:

Accepted file formats: csv, xlsx and xls.


Content Format

Your table should contains:

  1. DateTime in ISO (see below)
  2. Relative Humidity in %
  3. Temperature in F

DateTime is ISO time format (YYYY-MM-DDThh:mm:ss), for example: 0 hour time on 2019-06-01 would be expressed as 2019-06-01T00:00:00
